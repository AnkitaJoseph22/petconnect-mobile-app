# 🐾 PetConnect – Pet Adoption Platform

PetConnect is a full-stack web application that connects pet buyers, sellers, veterinarians, and administrators on a single platform to simplify the pet adoption process.

## Features

* Multi-role system (Buyer, Seller, Veterinarian, Admin)
* Secure authentication using JWT
* Pet listings and adoption management
* Payment gateway and geolocation integration

## Tech Stack

React, Node.js, Express, MongoDB, JWT, REST APIs

This repository contains my personal copy of the PetConnect project, originally developed as part of a collaborative academic project and presented at the **Science Exhibition 2025, Christ University**.
